# AnimeApp
This is a clean architecture app example built with

- MVVM
- Coroutines
- Extension Functions
- Dagger Hilt
- Retrofit
- Navigation Components
- Coroutines Flow
- ViewBinding
- Use Case
- Repository
- Coil
- ViewModel,LiveData

Including test cases built with

- MockWebServer
- Mockito
- Truth
- Turbine
